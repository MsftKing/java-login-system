# Java Login System

## Overview
This project is a simple Java-based console login system that allows user to register, log in, and validate credentials. Demonstrating fundamental programming concepts such as user input handling, conditional logic, and data storage using Java collections.

## Features 
- User registration (username & password)
- User login authentication
- Password validation
- Duplicate username prevention
- Console-based menu system
- Loop-based program flow

## Technologies Used 
- Java
- Scanner (user input)
- HashMap (user storage)

## How it works 
1. Program starts and displays a menu
2. User selections an option:
     - Register: Creates a new account
     - Login: Checks Credentials
     - Exit: Closes program
3. User data is stored in memory using a HashMap
4. Login checks username and password match before granting access

## Future Improvements

- store user data in files
- Encrypt passwords using hashing
